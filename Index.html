pygame.init()

11. Lägga till stöd för handkontroller

            

Om du vill spela med en handkontroll i ditt fightingspel i Pygame, kan du använda Pygames inbyggda joystick-modul. Här är en enkel guide för att komma igång.

                         

11.1) Aktivera Stöd för Handkontroller

            

För att Pygame ska känna igen din handkontroll, måste du initialisera joystick-modulen:

            
import pygame

# Initialisera pygame och joysticks
pygame.init()
pygame.joystick.init()

# Kolla om det finns någon ansluten handkontroll
if pygame.joystick.get_count() > 0:
    joystick = pygame.joystick.Joystick(0)
    joystick.init()
    print(f"Handkontroll ansluten: {joystick.get_name()}")
else:
    print("Ingen handkontroll hittades.")
                         

11.2) Lägga till Handkontroll-Styrning i Spelet

            

För att använda handkontrollen för att styra en spelare, kan du lägga till kod i din event-loop:

            
for event in pygame.event.get():
    if event.type == pygame.QUIT:
        run = False
    elif event.type == pygame.JOYAXISMOTION:
        if event.axis == 0:  # Vänster spak, horisontell rörelse
            if event.value < -0.5:
                print("Går vänster")
            elif event.value > 0.5:
                print("Går höger")
    elif event.type == pygame.JOYBUTTONDOWN:
        if event.button == 0:  # Exempel: X-knappen på en Xbox-kontroll
            print("Hoppa")
                         

11.3) Testa och Anpassa

            

Koppla in din handkontroll och testa styrningen. Om du vill anpassa knapparna, använd print(event.button) för att se vilket knappnummer som trycks och justera koden därefter.

                         

Nu kan du använda en handkontroll för att spela ditt fightingspel i Pygame!

Vanliga Frågor (FAQ)

            
                Spelet startar inte, vad gör jag?                 

Kontrollera att du har installerat Pygame med pip install pygame.

            
            
                Varför rör sig inte min spelare?                 

Se till att du har rätt tangentbordshantering och att pygame.key.get_pressed() används korrekt.

            
            
                Ljudet fungerar inte, vad kan vara fel?                 

Dubbelkolla att ljudfilerna finns på rätt plats och att Pygame Mixer är initierad.

            
            
                Hur kan jag lägga till fler animationer?                 

Lägg till fler sprites i en spritesheet och justera animationerna i din Fighter-klass.

            
            
                Kan jag använda andra bilder och ljud?                 

Ja! Byt ut filerna i assets/-mappen och uppdatera din kod.
